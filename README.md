# McStreamy

## What is it ?
A library for dealing with streams in .Net. Contains wrappers, extension methods, stream implementations etc.

## Supported Platforms
Currently;

* .Net Framework 4.0
* Silverlight 5.0
* NetStandard 1.0
    * .Net Framework 4.5+
    * Windows Phone Silverlight (8.0+) 
    * Xamarin.iOS 
    * Xamarin.Android
    * WinRT
    * UWP 10+
